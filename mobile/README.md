# Bornblix mobile (Expo)

Expo Router + NativeWind + Bun.

## Setup

Install dependencies (from repo root or this folder):

```bash
bun install
```

## Run

```bash
bunx expo start
```

Clear Metro cache after config changes:

```bash
bunx expo start --clear
```

From monorepo root:

```bash
bun run mobile
bun run mobile:clear
```

## Reset starter (optional)

```bash
bun run reset-project
```
